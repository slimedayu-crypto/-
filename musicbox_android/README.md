# MusicBox Pro — Android Edition

Портированная на Kivy версия плеера для сборки в APK через Buildozer.
Логика (yt-dlp поиск/загрузка, метаданные mutagen, плейлисты/история в SQLite)
сохранена; интерфейс переписан с tkinter на Kivy, т.к. tkinter не работает на Android.

## Что реализовано
- Локальная библиотека: автоскан папки `Music/MusicBoxPro`, чтение тегов (mutagen), поиск/фильтр по названию и исполнителю
- Поиск по YouTube и загрузка лучшего аудиопотока (yt-dlp), индикатор прогресса с процентами и скоростью
- Плеер: play/pause/next/prev/seek, история прослушиваний (SQLite)
- Обложки: извлекаются из ID3-тегов (mp3), либо подтягиваются онлайн через iTunes Search API после загрузки с YouTube, кэшируются в `_covers/`
- Тексты песен: автозагрузка через lyrics.ovh API, показ во всплывающем окне на экране плеера
- Плейлисты: создание, удаление, добавление треков (кнопка "+" на каждом треке), отдельный экран плейлиста
- Лайки/избранное: кнопка на каждом треке и на экране плеера
- Поиск дубликатов: группировка по хешу файла, удаление прямо из списка
- Статистика библиотеки: количество треков, избранных, плейлистов, суммарная длительность, топ-5 прослушиваемых треков
- 4 темы оформления (Dark/Purple/Green/Sunset), сохраняются между запусками

## Важное ограничение
На Android обычно недоступен бинарник `ffmpeg`, поэтому загрузка с YouTube
сохраняет аудио "как есть" (m4a/webm/opus) без конвертации в mp3 — это
по-прежнему воспроизводится плеером, просто без перекодирования.

## Как собрать APK — самый простой способ (бесплатно, в облаке)

1. Создайте новый репозиторий на GitHub (можно приватный).
2. Залейте туда все файлы из этой папки **как есть**, сохранив структуру
   (важно сохранить папку `.github/workflows/build.yml`).
   ```bash
   git init
   git add .
   git commit -m "MusicBox Pro Android"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГИН/musicboxpro.git
   git push -u origin main
   ```
3. Зайдите в репозиторий на GitHub → вкладка **Actions**. Сборка запустится
   автоматически (или нажмите "Run workflow" вручную). Первая сборка идёт
   20–40 минут (скачивается Android SDK/NDK), следующие — быстрее благодаря кэшу.
4. Когда сборка завершится зелёной галочкой — откройте её, внизу страницы
   будет **Artifacts → musicboxpro-apk**. Скачайте zip, внутри — `.apk`.
5. Перекиньте APK на телефон и установите (разрешите "установку из
   неизвестных источников" в настройках Android).

## Локальная отладка на ПК перед сборкой (необязательно)
```bash
pip install kivy yt-dlp mutagen requests plyer
python main.py
```

## Дальнейшие доработки
- Иконку/заставку — положите `icon.png` (512x512) и `presplash.png` в корень,
  пропишите пути в `buildozer.spec` (`icon.filename`, `presplash.filename`).
- Для релизной (подписанной) сборки под публикацию в Google Play потребуется
  `buildozer android release` + ключ подписи — это отдельный шаг, не нужен
  для личного использования.
